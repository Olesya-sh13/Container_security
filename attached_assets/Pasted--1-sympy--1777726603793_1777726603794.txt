Описание алгоритма для второй части общего задания
1.Импортировать библиотеку sympy для символьных вычислений.
2.Объявить символьные переменные: k, T, C.
3.Задать начальную стоимость C_ost = 100000.
4.Установить начальное значение накопленной амортизации Aj = 0.
5.Создать пустые списки для хранения результатов: Am_lst_2 (амортизация по годам) и C_ost_lst_2 (остаточная стоимость по годам).
6.Выполнить цикл для 5 лет (i от 1 до 5):
Вычислить годовую амортизацию по формуле Am = k * (1/T) * (C - Aj), где C = 100000, T = 5, k = 2.
Обновить остаточную стоимость: C_ost = C_ost – Am.
Добавить полученное значение Am с округлением до 2 знаков в список Am_lst_2.
Обновить накопленную амортизацию: Aj = Aj + Am.
Добавить текущее значение C_ost с округлением до 2 знаков в список C_ost_lst_2.
7.Вывести списки Am_lst_2 и C_ost_lst_2.
Описание алгоритма для третьей части общего задания
1.Объявить символьные переменные: k, T, C, L.
2.Установить значения переменных: C = 30000, T = 8, L = 0, k = 2.
3.Присвоить начальное значение остаточной стоимости: C_ost = 30000.
4.Создать пустые списки для накопления результатов: Am_lst – для годовых сумм амортизации и C_ost_lst – для остаточной стоимости на конец каждого года.
5.Выполнить цикл по годам от 1 до 8 (i от 1 до 8):
Вычислить годовую амортизацию по формуле линейного метода:
Am = (C – L) / T.
Подставить в формулу числовые значения C = 30000, L = 0, T = 8 и вычислить Am.
Обновить остаточную стоимость: C_ost = C_ost – Am.
Округлить полученное значение Am до двух знаков после запятой и добавить его в список Am_lst.
Округлить текущее значение C_ost до двух знаков и добавить его в список C_ost_lst.
6.Вывести списки Am_lst и C_ost_lst.
7.Инициализировать переменную накопленной амортизации: Aj = 0.
8.Создать пустые списки: Am_lst_2 – для амортизации по годам, C_ost_lst_2 – для остаточной стоимости на конец года.
9.Выполнить цикл по годам от 1 до 8 (i от 1 до 8):
Вычислить амортизацию по формуле метода уменьшаемого остатка:
Am = k * (1 / T) * (C – Aj).
Подставить в формулу числовые значения C = 30000, T = 8, k = 2.
Уменьшить остаточную стоимость: C_ost = C_ost – Am (с подстановкой числовых значений).
Округлить Am до двух знаков и добавить в список Am_lst_2.
Обновить накопленную амортизацию: Aj = Aj + Am.
Округлить C_ost до двух знаков и добавить в список C_ost_lst_2.
10.Вывести списки Am_lst_2 и C_ost_lst_2.
